## MERN chatrooms

to run this locally follow these steps:
  - clone this repo
  - ```$ npm run install-all```
  - ```$ touch server/.env && echo 'SECRET=whateveryouwant' > server/.env```
  - ```$ mongod```
  - ```$ npm start```


Created with 
  - node v 11.2.0
